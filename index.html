Data Sources
URA Official Data Primary

Official transaction records from the Urban Redevelopment Authority. Updated every Tuesday and Friday. Covers all private residential property transactions.

API: eservice.ura.gov.sg/maps/api
Data: data.gov.sg

Property Portals Secondary

Supplementary data from EdgeProp, 99.co, and PropertyGuru. Includes additional listing details and analytics not in URA data.

EdgeProp | 99.co | PropertyGuru

How to Update Data

Run the Python scripts in /scripts to fetch fresh data: 

# URA API (needs access key)
python scripts/fetch_ura_data.py

# Import portal CSV exports
python scripts/fetch_portal_data.py import data.csv --type cluster
